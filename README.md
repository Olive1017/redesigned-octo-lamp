# OCR 照片识别与归档工具

桌面程序：选择装着已带水印物流照片的本地文件夹，调用 OCR/大模型逐张识别，按车牌分组成「订单」，边识别边归档到文件夹（第一阶段）。

## 目录结构

```
config.py            # 配置文件（OCR 端点、鉴权、拼图参数等）
domain/
  enums.py           # 标识、订单状态枚举
  photo.py           # 照片数据模型
  order.py           # 订单数据模型
services/
  recognizer.py      # 识别器：HTTP POST 调用 + 解析
  validator.py       # 校验器：五类齐全校验（第二阶段占位）
  collager.py        # 拼图器：Pillow 二合一/三合一（第二阶段占位）
  writer.py          # 输出器：归档 + 收尾定名
  uploader.py        # 上传器：RPA 接口（第二阶段占位）
pipeline.py          # 流水线编排（第一阶段）
gui/
  main_window.py     # 主窗口（PySide6）
  worker.py          # 后台任务线程（QThread）
main.py              # 启动程序
requirements.txt     # 依赖列表
```

## 使用说明

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 修改 `config.py` 中的 OCR 服务配置：
   - 填写正确的 OCR_URL、鉴权信息、MODEL_ID 等

3. 运行程序：
   ```bash
   python main.py
   ```

4. 操作步骤（第一阶段）：
   - 点击「选择文件夹」选择装照片的文件夹
   - 点击「开始归档」开始识别和归档
   - 查看结果：已归档订单和待定/无车牌的清单

## 第一阶段功能

### 输出结构

归档完成后，`output/` 目录下：
```
output/
  粤CD0870_202501001/
    原文件名1.jpg
    原文件名2.jpg
    ...
  赣GM5690_/
    原文件名.jpg
  _待定/
    无车牌_原文件名.jpg
    识别失败_原文件名.jpg
```

### 处理逻辑

- **边识别边归档**：识别一张，立即按车牌归档到对应文件夹
- **无车牌**：放入 `_待定/` 文件夹，保留原文件名
- **收尾定名**：处理完成后，对有交货单号的订单，重命名文件夹为 `{车牌}_{交货单号}`
